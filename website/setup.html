BASIC INITIALIZATION

The basic steps for initializing Pygame involve importing the library, initializing it, and setting up the display surface.


  import pygame
  import sys
  
  # Initialize Pygame
  pygame.init()
  
  # Set up the display
  screen = pygame.display.set_mode((800, 600))
  pygame.display.set_caption("Basic Pygame Setup")
  
  # Main loop
  running = True
  while running:
      for event in pygame.event.get():
          if event.type == pygame.QUIT:
              running = False
      
      # Fill the screen with a color
      screen.fill((0, 0, 0))
      
      # Update the display
      pygame.display.flip()
  
  # Quit Pygame
  pygame.quit()
  sys.exit()

This example demonstrates the basic setup process in Pygame: initializing the library, creating a window, and running a loop to keep the window open. This forms the foundation of any Pygame application.

ADVANCED SETUP OPTIONS

Pygame offers various advanced setup options, such as setting the frame rate, handling different screen resolutions, and adding custom icons or background music.


import pygame
import sys
import os

pygame.init()

screen = pygame.display.set_mode((400, 400))
pygame.display.set_caption("Advanced Pygame Setup")

clock = pygame.time.Clock()

icon_path = 'images/download.png'
if os.path.isfile(icon_path):
    icon = pygame.image.load(icon_path)
    pygame.display.set_icon(icon)

music_path = 'music/sample.wav'
if os.path.isfile(music_path):
    pygame.mixer.music.load(music_path)
    pygame.mixer.music.play(-1)

running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    
    screen.fill((0, 0, 0))
    pygame.display.flip()
    clock.tick(60)

pygame.quit()
sys.exit()

The code initializes Pygame, sets up a window with a custom icon, and plays background music. It runs a loop to keep the window open, updates the display with a black screen, and maintains a 60 FPS frame rate.To run this code correctly make sure you edit the paths to image and music depending on your project settings
